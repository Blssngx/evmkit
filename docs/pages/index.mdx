## The EVM Kit Stack

EVM Kit covers all aspects of your web3 application, from smart contracts to the front end.

### Smart Contracts

- **Solidity**: The language of choice to write the logic for your smart contracts.
- **Hardhat**: A development environment to compile, test, and debug your smart contracts locally.
- **thirdweb ContractKit**: Gas-optimized, battle-tested smart contracts for common use cases like tokens & NFTs.
- **OpenZeppelin Contracts**: A set of smart contracts for utilities and security.

### Front End

- **Next.js**: Optimized framework for creating applications using React.
- **TypeScript**: Type-safety for a better developer experience and safer code.
- **thirdweb React SDK**: React hooks and UI components for wallet and smart contract interactions.
- **thirdweb Auth**: React hooks to authenticate users with the Sign In With Ethereum standard.
- **thirdweb Storage**: React hooks to upload to and download from decentralized storage.
- **BYO UI Library**: Use any UI library you prefer. We have intentionally left this out of the stack.

### Deployment

- **thirdweb Deploy**: Simple UI for deploying your smart contract to any EVM-compatible network.
- **Vercel**: Seamless deployment of your Next.js application.
